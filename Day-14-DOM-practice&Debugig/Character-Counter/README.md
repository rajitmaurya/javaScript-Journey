# Character Counter 🔢

A simple JavaScript project that counts characters in real-time as the user types in an input field.

---

## 🚀 Features
- Live character count
- Updates instantly while typing
- Character limit (20 characters)
- Visual feedback when limit is reached

---

## 📚 Concepts Used
- DOM manipulation
- Event handling (`input` event)
- `.value` and `.length`
- Conditional statements
- String methods (`slice()`)

---

## 🛠 How It Works
- User types in the input field
- JavaScript listens to the `input` event
- Counts characters using `.length`
- Displays count on screen
- Limits input using `slice()` if it exceeds max length

---

## 📁 Files
- `index.html` – Structure of the app
- `script.js` – Logic for counting characters
